# The Project Archives of Marisol
Project Archives is a personal portfolio website designed to resemble a digital library, where each project is presented as a curated “archive” entry. Instead of a traditional portfolio layout, this site emphasizes exploration and organization, allowing visitors to browse through projects as if they were navigating a collection of stored knowledge.

The goal of this project is to combine creativity with functionality by blending a thematic design (library/archive aesthetic) with modern web development practices. It highlights not only completed work but also the process and structure behind each project. 

## Purpose
This project was created as part of a CMPA 3403 course to showcase personal and academic work in a unique and engaging format. It is intended for potential employers, collaborators, and instructors who want to explore projects in a more immersive and visually structured way.

Additionally, it serves as a demonstration of front-end development skills, design thinking, and the ability to build a cohesive user experience around a central concept.

## Tools and Technology
- HTML – Provides the structural foundation for organizing content in a clear and semantic way
- CSS – Used to create the library-inspired design, layout, and visual styling
- JavaScript – Adds interactivity and dynamic behavior to enhance user experience
- GitHub – Used for version control and project management
- GitHub Pages – Chosen for simple and efficient deployment of the live site
- Visual Studio Code – Chosen as the primary code editor for its extensions, debugging tools, and developer-friendly environment
